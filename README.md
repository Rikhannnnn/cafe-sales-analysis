# Cafe Sales Analysis

## Project Overview
Proyek ini bertujuan untuk menganalisis data penjualan sebuah Cafe untuk mengidentifikasi pola penjualan, produk terlaris, dan memberikan strategi bisnis guna meningkatkan omzet.

## Dataset
Dataset yang digunakan berisi data transaksi kotor yang meliputi:
- Transaction Date
- Item & Quantity
- Total Spent
- Location & Payment Method

## Tools & Libraries
- **Python**: Bahasa pemrograman utama.
- **Pandas**: Untuk manipulasi dan pembersihan data.
- **Seaborn/Matplotlib**: Untuk visualisasi data.

## Key Insights
1. **Produk Terlaris**: Juice adalah item dengan kuantitas penjualan tertinggi (1.124 item).
2. **Produk Bernilai Tinggi**: Salad memiliki rata-rata nilai transaksi tertinggi ($14.79).
3. **Waktu Ramai**: Penjualan memuncak pada hari **Minggu & Senin**.
4. **Metode Bayar**: Pelanggan Juice lebih menyukai pembayaran via **Digital Wallet**.

## Business Recommendations
- **Bundling Strategy**: Menggabungkan "Kopi" (Traffic Maker) dengan "Cake" untuk meningkatkan penjualan Cake.
- **Weekend Promo**: Memberikan cashback khusus pengguna Digital Wallet di hari Jumat-Minggu.
- **SOP Perbaikan Data**: Mewajibkan kasir mengisi kolom pembayaran untuk mengurangi data 'Unknown'.
